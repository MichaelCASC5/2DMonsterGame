2D Monster Maze Game

Welcome to 2D Monster Maze Game

Navigate through a monster-infested maze and sruvive!

Game features

  • Randomly Generated Maze

    Every Game starts with a new maze layout

  • Player Controls

    Move using WASD keys, shoot lasers with the F key. rotate using C and V keys. Pause the game with P key

  • Enemies and Power-ups

    Navigate thorugh the maze, kill enemies, collect powerups and SURVIVE

  • Levels

    Developed using the SFML 2D graphics library in C++

Frequently Asked Questions

  • How do I begin?

    Enter "make" into the WSL terminal. Ensure you have SFML 2D graphics library installed

  • What are the Conctrols?

    Use WASD keys to move, and F key to shoot lasers

  • What happens if a enemy tocuhes the player?

    The Player looses a heart. After loosing 3 hearts, the game is over

  • How many levels are there?

    There are 2 levels. The first level spawns 4 enemies, and the scond level spawns 8 enemies.

  • What languages and libraries are used?

    The game is built in c++, using the SFML 2D graphics Library
